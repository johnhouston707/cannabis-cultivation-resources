## Quick Reference: HLVd Prevention Protocol

1. **Test all incoming clones** before introducing them to your facility (RT-PCR, not visual inspection)
2. **Quarantine new genetics** for 2-4 weeks with a PCR test before integration
3. **Sterilize tools** between every cut (10% bleach or 70% IPA, 30 seconds minimum)
4. **Test mothers quarterly** on a rolling 90-day schedule
5. **Replace mothers** from tissue culture every 12-18 months
6. **Never share tools** between facilities or between genetic lines

## Quick Reference: Clone Rooting Checklist

| Factor | Optimal Range |
|--------|--------------|
| Humidity | 80-95% (dome or mist) |
| Temperature | 75-80°F (24-27°C) |
| Light | 100-200 PPFD (18/6) |
| Rooting hormone | IBA 0.1-0.3% (gel or powder) |
| Medium | Rockwool, rapid rooters, or peat plugs |
| Cut angle | 45° below a node |
| Stem length | 4-6 inches |
| Expected root time | 7-14 days |
