# California Cannabis Licensing Guide for Cultivators

## Getting Your Cannabis Cultivation License in California

Starting a legal cannabis cultivation operation in California requires navigating multiple regulatory layers. This guide covers the essentials.

## License Types

### Department of Cannabis Control (DCC) License Types

| License Type | Canopy Size | Indoor/Outdoor |
|-------------|-------------|----------------|
| Specialty Cottage | Up to 25 plants | Indoor |
| Specialty Cottage | Up to 2,500 sq ft | Outdoor |
| Specialty Small | 5,001-10,000 sq ft | Indoor |
| Specialty Small | 2,501-10,000 sq ft | Outdoor |
| Small | 10,001-22,000 sq ft | Mixed |
| Medium | 10,001-22,000 sq ft | Indoor |
| Large | 22,001+ sq ft | All |

### Nursery License (Type 4)

If you plan to sell clones or seeds commercially, you need a **Type 4 Nursery License** specifically. This allows:
- Propagation of cannabis plants
- Sale of immature plants (clones) and seeds
- Maintenance of mother plants for commercial distribution

## Local Authorization First

California requires **local approval before state licensing**. This means:
1. Check if your city/county allows commercial cannabis cultivation
2. Apply for local permits (conditional use permit, land use permit)
3. Secure local authorization letter
4. Then apply to the DCC for state license

## Key Requirements

### Facility
- Secure, enclosed space with restricted access
- Security cameras (24/7 recording, 90-day retention)
- Track-and-trace system (METRC) integration
- Proper waste management plan
- Water source documentation

### Testing
- All harvested cannabis must pass state-mandated testing
- Nurseries must maintain pest-free environments
- [HLVd testing](https://www.primecutsnursery.com/hplvd-testing) is not yet state-mandated but is industry best practice

### Financial
- Application fees: $1,000-15,000+ depending on license type
- Annual license fees: $2,500-75,000+ depending on canopy size
- Local permit fees vary widely by jurisdiction
